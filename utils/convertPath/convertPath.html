路径转换

convertPath ​

将 Windows 反斜杠路径转换为斜杠路径

基础用法

​

Demo1
windows路径：Documents\newFolder
转换后的路径：Documents/newFolder
Demo2
windows路径：C:\Documents\newFolder\test.js
转换后的路径：C:/Documents/newFolder/test.js
查看代码
vue
<script setup lang="ts">
import { ref } from "vue";
import { convertPath, clone } from "@pureadmin/utils";

let path1 = ref("Documents\\newFolder");
let path2 = ref("C:\\Documents\\newFolder\\test.js");
</script>

<template>
  <naive-theme>
    <n-space className="flex items-center mt-2">
      <n-card title="Demo1" embedded :bordered="false">
        <n-gradient-text type="info">
          windows路径：{{ clone(path1) }}
        </n-gradient-text>
        <br />
        <n-gradient-text type="error">
          转换后的路径：{{ convertPath(path1) }}
        </n-gradient-text>
      </n-card>
    </n-space>

    <n-space className="flex items-center mt-2">
      <n-card title="Demo2" embedded :bordered="false">
        <n-gradient-text type="info">
          windows路径：{{ clone(path2) }}
        </n-gradient-text>
        <br />
        <n-gradient-text type="error">
          转换后的路径：{{ convertPath(path2) }}
        </n-gradient-text>
      </n-card>
    </n-space>
  </naive-theme>
</template>

接收一个参数 path ，返回值类型 string
